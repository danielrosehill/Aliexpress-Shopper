# /aliexpress-similar

Find similar items to the AliExpress listing the user is on (or a URL they paste).

Usage: `/aliexpress-similar` or `/aliexpress-similar <listing-url>`

On-demand equivalent of the bundled **Find Similar** userscript's per-card button.
